# Story 35.5: Shoji Theme — Door-Like Proportions

## Status: Done (PR #237)

**Epic:** 35 — Door Visual Appearance — Door-Like Proportions
**Depends on:** Story 35.1 (PR #226)

## User Story

As a user,
I want the Shoji theme to render doors with door-like proportions using DoorAnatomy,
So that Shoji doors visually read as Japanese sliding doors with proper structural layout.

## Acceptance Criteria

**Given** the Shoji theme render function
**When** rendering a door with height >= MinHeight (14)
**Then:**

- AC1: Door is rendered in portrait orientation using DoorAnatomy row positions
- AC2: A lattice bar is rendered at ~45% of door height (PanelDivider) with cross junction (┼/╋)
- AC3: A handle element is rendered on the right side at ~60% of door height
- AC4: A threshold line is rendered below the bottom rail
- AC5: Content is placed starting at ContentStart row
- AC6: Top rail uses ┬─┬ (unselected) / ┳━┳ (selected) and bottom rail uses ┴─┴ / ┻━┻

**Given** the Shoji theme in compact mode (height < MinHeight or height == 0)
**When** rendering a door
**Then:**

- AC7: The existing layout is used (backward compatible)

**Given** golden file tests for the Shoji theme
**When** tests run at standard height (16 rows) and tall height (24 rows)
**Then:**

- AC8: Golden files match expected door-like output
- AC9: Both selected and unselected states are tested
