# Story 35.4: Sci-Fi Theme — Door-Like Proportions

## Status: Done (PR #236)

**Epic:** 35 — Door Visual Appearance — Door-Like Proportions
**Depends on:** Story 35.1 (PR #226)

## User Story

As a user,
I want the Sci-Fi theme to render doors as bulkhead/airlock panels with door-like proportions,
So that Sci-Fi doors clearly read as spaceship doors.

## Acceptance Criteria

**Given** the Sci-Fi theme render function
**When** rendering a door with height >= MinHeight (14)
**Then:**

- AC1: Door uses portrait orientation with double-line outer frame (`╔═╗║╚═╝`)
- AC2: A bulkhead divider (`╞═════════╡`) separates upper and lower sections at ~45% height
- AC3: An access panel handle (`◈──┤`) is rendered on the right side at ~60% height
- AC4: The `[ACCESS]` label is rendered in the lower panel area
- AC5: A floor grating line (`▓▓▓`) is rendered below the bottom frame
- AC6: Shade rails (`░` / `▓`) remain on sides

**Given** the Sci-Fi theme in compact mode (height < MinHeight or height == 0)
**When** rendering a door
**Then:**

- AC7: The existing card-style layout is used (backward compatible)

**Given** golden file tests for the Sci-Fi theme
**When** tests run at minimum, standard, and wide heights
**Then:**

- AC8: Golden files match expected door-like output
