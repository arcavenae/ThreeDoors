# Story 80.5: Stream Deck+ Encoder Support and Polish

**Epic:** 80 — Stream Deck Plugin (Elgato SDK)
**Status:** Not Started
**Priority:** P2
**Estimate:** M (encoder actions, LCD rendering, final polish)
**Repo:** `ArcavenAE/threedoors-streamdeck`

## Context

Stream Deck+ devices have rotary encoders with LCD touchscreen strips. These can display richer information (session timer, stats) and provide alternative interaction patterns (rotate to browse tasks). This story also covers final polish: icon design, pre-built profiles, and Marketplace preparation.

**Research:** [Stream Deck Research](../../_bmad-output/planning-artifacts/streamdeck-research.md) — Section 2 (Stream Deck+ Encoder Support), Section 3 (Marketplace Publishing).

## Acceptance Criteria

- [ ] `com.arcavenae.threedoors.session` encoder action registered with `controllers: ["Encoder"]`
- [ ] Encoder LCD shows session stats: tasks completed, session duration, current streak
- [ ] `dialRotate` cycles through stats displays (completed → duration → streak → repeat)
- [ ] `dialDown` (press) resets/starts a new session timer
- [ ] Encoder LCD updates via `setFeedback` with formatted text and progress indicators
- [ ] `com.arcavenae.threedoors.browse` encoder action for task browsing
- [ ] `dialRotate` on browse encoder scrolls through available tasks (beyond the 3 doors)
- [ ] `dialDown` on browse encoder selects the displayed task as current
- [ ] Browse LCD shows task text and metadata (type, effort)
- [ ] Final icon set: 8 icons (3 doors, complete, skip, snooze, refresh, status) in 72x72 and 144x144 (@2x)
- [ ] Pre-built Stream Deck profile exported as `.streamDeckProfile` with optimal 3-door layout
- [ ] `streamdeck pack` produces valid `.streamDeckPlugin` installer
- [ ] `streamdeck validate` passes with zero warnings
- [ ] Plugin README includes gallery screenshots for Marketplace submission
- [ ] Tests: encoder rotation handling, LCD content formatting, profile validity

## Technical Notes

### Encoder LCD API

```typescript
// setFeedback updates the encoder's LCD strip
action.setFeedback({
  title: "Completed",
  value: "7 tasks",
  indicator: { value: 70, bar_bg_c: "0:#555555", bar_fill_c: "0:#00FF00" }
});
```

### Pre-Built Profile

Stream Deck profiles define button layouts. A "ThreeDoors" profile with:
- Row 1: Door 1 | Door 2 | Door 3 | Refresh
- Row 2: Complete | Skip | Snooze | Status

Exported via Stream Deck app → Profiles → Export.

### Marketplace Preparation

This story brings the plugin to Marketplace-ready state but does NOT submit. Submission is a manual human process that requires Maker Console registration. The story ensures all technical prerequisites are met.
