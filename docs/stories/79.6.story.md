# Story 79.6: Homebrew Service Integration and launchd Support

**Epic:** 79 — ThreeDoors Daemon/Server Mode
**Status:** Not Started
**Priority:** P2
**Estimate:** S (Homebrew formula update, launchd plist, documentation)

## Context

For the daemon to be useful in practice, it needs to start automatically and stay running. macOS users expect `brew services start threedoors` to manage background processes. This story adds Homebrew service support and a fallback launchd plist for non-Homebrew installs.

**Research:** [Stream Deck Research](../../_bmad-output/planning-artifacts/streamdeck-research.md) — Section 4 (Homebrew Compatibility).

## Acceptance Criteria

- [ ] Homebrew formula updated with `service` stanza for `threedoors serve`
- [ ] `brew services start threedoors` launches the daemon in the background
- [ ] `brew services stop threedoors` cleanly stops the daemon
- [ ] Service log output goes to `~/Library/Logs/threedoors-daemon.log` (Homebrew convention) or `$(brew --prefix)/var/log/`
- [ ] Standalone launchd plist template at `docs/examples/com.arcavenae.threedoors-daemon.plist`
- [ ] Plist uses `KeepAlive: true` for automatic restart on crash
- [ ] `threedoors serve --install-service` generates and loads the launchd plist (non-Homebrew users)
- [ ] `threedoors serve --uninstall-service` unloads and removes the launchd plist
- [ ] Documentation in README or `docs/` explaining daemon setup for both Homebrew and manual users
- [ ] Tests: plist generation with correct paths, service install/uninstall commands

## Technical Notes

### Homebrew Service Stanza

```ruby
service do
  run [opt_bin/"threedoors", "serve"]
  keep_alive true
  log_path var/"log/threedoors-daemon.log"
  error_log_path var/"log/threedoors-daemon.log"
end
```

### launchd Plist

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "...">
<plist version="1.0">
<dict>
  <key>Label</key>
  <string>com.arcavenae.threedoors-daemon</string>
  <key>ProgramArguments</key>
  <array>
    <string>/usr/local/bin/threedoors</string>
    <string>serve</string>
  </array>
  <key>KeepAlive</key>
  <true/>
  <key>StandardOutPath</key>
  <string>~/Library/Logs/threedoors-daemon.log</string>
</dict>
</plist>
```
