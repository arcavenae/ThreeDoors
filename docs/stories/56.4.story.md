## Story

As a user,
I want the upper and lower door panels to have differentiated background colors,
So that the two-panel door structure is visually distinct and adds depth to the door surface.

## Background

Story 56.1 adds a uniform background fill to all interior rows. This story refines that by making the upper panel (above the divider) slightly lighter than the lower panel (below the divider), creating a subtle two-tone effect. Real doors often have differentiated upper/lower panels вҖ” this visual cue reinforces the "door" metaphor.

## Acceptance Criteria

**Given** a door is rendered in any of the 8 themes
**When** interior rows above the panel divider are displayed
**Then** the background color is `Fill` (the theme's standard interior color)

**Given** a door is rendered in any of the 8 themes
**When** interior rows below the panel divider are displayed
**Then** the background color is `FillLower` (darker than `Fill`)

**Given** the panel divider row
**When** determining which rows use which background
**Then** the divider row itself uses `FillLower` background (it belongs to the lower section)

**Given** each of the 8 themes
**When** `FillLower` colors are applied
**Then** each theme uses values from the research artifact palette:
- Classic: `#080820` / `17` / `0`
- Modern: `#080808` / `232` / `0`
- Sci-Fi: `#061425` / `17` / `0`
- Shoji: `#141005` / `233` / `0`
- Winter: `#060a14` / `232` / `0`
- Spring: `#061408` / `232` / `0`
- Summer: `#141005` / `233` / `0`
- Autumn: `#140a05` / `232` / `0`

**Given** a door at minimum width and minimum height
**When** rendered with panel zone shading
**Then** both zones are visible and the transition is at the divider row

**Given** existing golden file tests
**When** the panel zone changes are applied
**Then** golden files are regenerated to match the new rendering
