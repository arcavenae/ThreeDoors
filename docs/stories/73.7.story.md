## Story

As a multiclaude operator,
I want a `/rollcall` command that shows all running agents with their status, type, and context usage,
So that I can quickly assess fleet health without checking tmux windows individually.

## Acceptance Criteria

- [ ] `scripts/rollcall.sh` gathers agent data from state.json, tmux, and process info
- [ ] `scripts/rollcall.sh --detail` adds memory usage and context token estimates
- [ ] `.claude/commands/rollcall.md` skill invokes the script and adds interpretation
- [ ] `just rollcall` recipe runs the shell script
- [ ] Script handles missing agents, dead processes, and missing JSONL files gracefully
- [ ] Script requires only `jq` and standard Unix tools (plus `python3` for `--detail` context parsing)
