## Story

As a user,
I want mood productivity correlations displayed as colored horizontal bar charts instead of plain text numbers,
So that I can instantly compare how productive I am in different moods through visual proportions.

## Background

The current `renderMoodCorrelations()` outputs plain text: `Focused: avg 4.2 tasks/session (12 sessions)`. This requires reading and mentally comparing numbers. Horizontal bar charts make the comparison instant â€” the longest bar is immediately obvious.

Each mood gets a distinct color: Focused=blue, Energized=yellow, Calm=green, Tired=gray. Bars use `â–ˆ` for filled and `â–‘` for empty portions, proportional to the average tasks per session.

## Acceptance Criteria

**Given** mood correlation data exists with multiple moods
**When** the mood panel renders
**Then** each mood displays a horizontal bar using `â–ˆ` (filled) and `â–‘` (empty) characters
**And** bar length is proportional to avg tasks per session relative to the highest mood
**And** each mood's bar is colored with its designated color
**And** the numeric average and session count appear after the bar

**Given** a mood has the highest average
**When** it renders
**Then** its bar fills 100% of the available width

**Given** no mood data exists
**When** the mood panel renders
**Then** the existing "Not enough mood data yet" message displays
**And** no empty bars are shown

**Given** the terminal is narrow (< 60 cols)
**When** bars render
**Then** bars use a shorter width appropriate for the compact layout
**And** labels are abbreviated if necessary

## Technical Notes

- Modify `renderMoodCorrelations()` in `insights_view.go`
- Bar fill: `strings.Repeat("â–ˆ", fillWidth) + strings.Repeat("â–‘", emptyWidth)`
- Style each bar with mood-specific color using `lipgloss.NewStyle().Foreground()`
- Mood colors: Focused=#60A5FA, Energized=#FBBF24, Calm=#34D399, Tired=#9CA3AF
- Bar width = available panel width minus label and value columns
- Max bar = 100% fill, others proportional
- Use `lipgloss.AdaptiveColor` for light/dark terminal support
