## Story

As a user,
I want the shadow gradient to be fine-tuned across terminal widths and all 8 themes,
So that shadows look correct at every width from minimum (no shadow) to wide (full 3-column gradient).

## Background

Story 56.3 implements the gradient shadow infrastructure. This story is the tuning and polish pass — verifying that shadow contrast ratios are perceptible across all 8 themes on dark backgrounds, adjusting gradient colors if needed, and adding comprehensive golden file tests at multiple widths to prevent visual regressions.

The research artifact specifies width thresholds:
- Width < minWidth + 2: no shadow
- Width >= minWidth + 2: 1-column shadow
- Width >= minWidth + 4: 2-column gradient
- Width >= minWidth + 6: 3-column gradient

This story validates those thresholds produce correct output at narrow (~48 cols), medium (~80 cols), and wide (~120 cols) terminal widths.

## Acceptance Criteria

**Given** each of the 8 themes
**When** rendered at narrow width (~48 cols, ~15-char doors)
**Then** no shadow is rendered
**And** background fill and bevel are still present

**Given** each of the 8 themes
**When** rendered at medium width (~80 cols, ~24-char doors)
**Then** 1-2 column shadow is rendered with visible gradient

**Given** each of the 8 themes
**When** rendered at wide width (~120 cols, ~38-char doors)
**Then** 3-column shadow gradient (▓▒░ or bg-colored) is rendered with maximum depth

**Given** the shadow gradient colors for each theme
**When** compared against a dark terminal background (#000000 to #1a1a2e)
**Then** the near-shadow column has at least 4:1 contrast ratio against the terminal background
**And** the far-shadow column fades to near-terminal-background levels

**Given** golden file tests for shadow rendering
**When** tests run at narrow, medium, and wide widths
**Then** golden files exist for each width tier per theme
**And** output matches the expected gradient

**Given** selected vs unselected doors
**When** rendered with shadow at the same width
**Then** selected doors have wider shadows (1 extra column) than unselected doors
