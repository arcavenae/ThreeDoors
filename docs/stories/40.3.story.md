## Story

As a user,
I want to see a rotating fun fact at the bottom of the insights dashboard drawn from my real session data,
So that I discover surprising personal patterns and feel celebrated for my activity.

## Background

ThreeDoors captures extensive behavioral data but never surfaces aggregate observations as conversational insights. A fun fact line transforms cold data into a personal, delightful moment — the "friend saying hey, look at what you did!" that SOUL.md describes.

The fun facts engine follows four mandatory content rules from party mode decision D-089:
1. Observe, don't prescribe
2. Celebrate totals, not rates
3. Frame gaps as potential
4. No decline comparisons

## Acceptance Criteria

**Given** the insights dashboard renders with sufficient data
**When** the view displays
**Then** a fun fact appears at the bottom, styled with a gold star prefix
**And** the fact references real data from the user's sessions

**Given** the user opens insights multiple times on the same day
**When** the view renders each time
**Then** the same fun fact is shown (deterministic daily rotation)

**Given** the user opens insights on a different day
**When** the view renders
**Then** a different fun fact is shown (rotated by day)

**Given** the fun facts pool is generated
**When** any fact is examined
**Then** it contains no words from the banned list: "declined", "dropped", "worse", "less", "failed", "missing", "behind", "overdue"
**And** it passes the "friend test" — a supportive friend would say this

**Given** insufficient data for any fun fact
**When** the fact pool is empty
**Then** a default welcoming fact is shown: "Every door you open is a step forward!"

## Technical Notes

- New file: `internal/core/fun_facts.go`
- New file: `internal/core/fun_facts_test.go`
- `FunFactGenerator` struct with `analyzer *PatternAnalyzer` and `counter *CompletionCounter`
- `Generate() string` returns a single fact for today
- `buildFactPool() []string` assembles all available facts from current data
- Selection algorithm: `day := time.Now().UTC().YearDay() + time.Now().UTC().Year()*366; idx := day % len(facts)`
- Fact templates (~15-20):
  - "You've opened {n} doors since your first session!"
  - "You complete the most tasks on {day}s"
  - "Your longest streak was {n} days"
  - "You pick the {position} door {pct}% of the time"
  - "{mood} is your power mood — avg {n} tasks!"
  - "You've completed {n} tasks in total — every one counts!"
  - etc.
- InsightsView calls `iv.funFactGen.Generate()` and renders with gold star styling
- Constructor: `NewFunFactGenerator(analyzer, counter)` — inject via InsightsView constructor
