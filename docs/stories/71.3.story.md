# Story 71.3: Update Docs, Tests, and Agent Definitions for Intel Removal

## Status: Done (PR #817)

## Epic

Epic 71: Drop Apple Intel (darwin/amd64) Builds

## References

- Issue: https://github.com/arcaven/ThreeDoors/issues/803
- Depends on: Stories 71.1, 71.2

## Story

As a project maintainer,
I want documentation, tests, and agent definitions to reflect that darwin/amd64 is no longer built,
So that users and agents have accurate information about supported platforms.

## Background

After Stories 71.1 and 71.2 remove darwin/amd64 from the build and release pipelines, several documentation files, test cases, and agent definitions still reference Intel Mac binaries. These need to be updated for consistency.

## Acceptance Criteria

**AC1: README installation tables drop Intel entries**
**Given** `README.md` lists "macOS (Intel)" in the pkg installer and binary download tables
**When** the Intel rows are removed
**Then** only Apple Silicon and Linux entries remain in both tables

**AC2: Docs-site installation page updated**
**Given** `docs-site/docs/getting-started/installation.md` mentions Intel support
**When** Intel references are removed
**Then** the page reflects Apple Silicon-only macOS support

**AC3: Architecture doc updated**
**Given** `docs/architecture/infrastructure-and-deployment.md` lists "darwin/amd64" in the cross-compile target list
**When** amd64 is removed from the target descriptions (lines ~40 and ~88)
**Then** only darwin/arm64 and linux/amd64 are listed

**AC4: Test table entries updated**
**Given** `internal/dist/codesign_test.go` has an "amd64" test case in `TestCodeSigner_SignMultipleBinaries`
**And** `internal/dist/pkg_builder_test.go` has an "amd64" test case in `TestPkgBuilder_Build_PerArchitecture`
**When** the amd64 test table entries are removed
**Then** only arm64 test cases remain
**And** `go test ./internal/dist/...` passes

**AC5: Release manager agent definition updated**
**Given** `agents/release-manager.md` lists "darwin-amd64" in the cross-compile description
**When** the reference is updated
**Then** only darwin-arm64 and linux-amd64 are listed

**AC6: All checks pass**
**Given** all documentation and test changes are made
**When** `just fmt`, `just lint`, `just test` are run
**Then** all pass with zero warnings

## Tasks
