**Status:** Done (PR #307)
**Priority:** P1
**Estimate:** M (2-4 hours)

## User Story

As a ThreeDoors user, I want to declare that a task depends on other tasks, so that the system understands which tasks must be completed before others can begin.

## Context

This is the foundational story for Epic 29. It adds the `DependsOn []string` field to the Task struct and creates a standalone `DependencyResolver` component with pure functions for checking unmet dependencies, detecting cycles, and finding newly unblocked tasks. This story covers FR110 (depends_on field) and FR115 (circular dependency detection).
