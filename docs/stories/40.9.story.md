## Story

As a user,
I want my stats dashboard colors to match my chosen door theme,
So that the entire ThreeDoors experience feels visually cohesive.

## Background

Phase 1 uses an independent color palette for the stats dashboard (blue-teal-yellow gradient, mood-specific bar colors). This works but feels disconnected from the door themes. Each door theme has a distinct personality (Classic=warm, Modern=cool, Sci-Fi=neon, Shoji=earth) that should extend to the stats view.

This story extends `ThemeColors` with stats-specific color fields and updates each theme to provide its own stats palette. The InsightsView receives the active theme and applies its colors to panel borders, hero numbers, sparkline gradients, and bar chart accents.

## Acceptance Criteria

**Given** the user has selected the Classic theme
**When** the insights view renders
**Then** panel borders, hero number, and sparkline gradient use Classic theme's warm amber palette

**Given** the user has selected the Sci-Fi theme
**When** the insights view renders
**Then** panel borders, hero number, and sparkline gradient use Sci-Fi theme's neon green/cyan palette

**Given** all four themes (Classic, Modern, Sci-Fi, Shoji)
**When** each theme's stats colors are applied
**Then** the stats view has a visually distinct personality matching the door theme
**And** all color combinations pass accessibility checks (sufficient contrast)

**Given** no theme is active (nil theme)
**When** the insights view renders
**Then** the Phase 1 independent palette (blue-teal-yellow) is used as fallback

## Technical Notes

- Extend `ThemeColors` struct in `internal/tui/themes/types.go` with:
  - `StatsAccent lipgloss.TerminalColor` — panel borders, hero number
  - `StatsGradientStart lipgloss.TerminalColor` — sparkline low end
  - `StatsGradientEnd lipgloss.TerminalColor` — sparkline high end
- Update each theme file (`classic.go`, `modern.go`, `scifi.go`, `shoji.go`) with theme-specific stats colors
- InsightsView receives theme via constructor: `NewInsightsView(analyzer, counter, theme)`
- In render functions, check for theme colors; fall back to independent palette if nil
- Theme personality mapping:
  - Classic: warm amber (#D97706) accent, amber-to-gold gradient
  - Modern: cool teal (#0D9488) accent, blue-to-teal gradient
  - Sci-Fi: neon green (#22C55E) accent, green-to-cyan gradient
  - Shoji: earth brown (#92400E) accent, clay-to-sand gradient
