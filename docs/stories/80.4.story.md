# Story 80.4: Connection Management and Status Display

**Epic:** 80 — Stream Deck Plugin (Elgato SDK)
**Status:** Not Started
**Priority:** P1
**Estimate:** S (status action, property inspector, wake handling)
**Repo:** `ArcavenAE/threedoors-streamdeck`

## Context

The plugin needs robust connection management — the daemon may not be running, may crash, or the system may sleep and wake. Users need a clear status indicator and a way to configure the connection.

## Acceptance Criteria

- [ ] `com.arcavenae.threedoors.status` action shows connection state on a button
- [ ] Status states: "Connected" (green icon), "Connecting..." (yellow icon), "Offline" (red icon)
- [ ] Status button shows session stats when connected (tasks completed today)
- [ ] Long-press on status button attempts reconnection
- [ ] `systemDidWakeUp` event triggers daemon reconnection attempt
- [ ] Property inspector for status action allows configuring daemon socket path
- [ ] Property inspector shows daemon version when connected
- [ ] Global settings store socket path (shared across all actions, not per-button)
- [ ] All actions gracefully degrade when daemon is offline (alert on press, "Offline" titles)
- [ ] Connection state changes propagate to all action instances (door titles update)
- [ ] Plugin startup attempts connection immediately, falls back to retry loop
- [ ] Tests: connection state machine, wake reconnect, global settings propagation

## Technical Notes

### Property Inspector

The property inspector is an HTML page rendered in Stream Deck's Chromium. It reads/writes `globalSettings` which all action instances share:

```html
<sdpi-item label="Daemon Socket">
  <sdpi-textfield setting="socketPath" placeholder="~/.threedoors/daemon.sock"></sdpi-textfield>
</sdpi-item>
<sdpi-item label="Status">
  <span id="daemon-version">Not connected</span>
</sdpi-item>
```

### Connection State Machine

```
Disconnected → Connecting → Connected
     ↑              |            |
     └──────────────┘            |
     └───────────────────────────┘
```

Transitions trigger icon/title updates on all visible actions.
