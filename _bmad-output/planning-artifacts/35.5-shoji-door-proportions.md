# Story 35.5 Planning Artifact — Shoji Theme Door-Like Proportions

## Adopted Approach: Row-by-row rendering using DoorAnatomy with Shoji aesthetics

**Rationale:** Follow the same pattern established by Classic (35.2), Modern (35.3), and Sci-Fi (35.4) — use `DoorAnatomy` to calculate structural row positions and render the Shoji theme row-by-row with:
- Top rail (┬─┬ / ┳━┳) at LintelRow
- Lattice bars at structural rows, content between ContentStart and PanelDivider
- Mid-cross bar with ┼/╋ junction at PanelDivider (maps naturally to Shoji's existing cross junction)
- Handle element (○) at HandleRow on the right side — matches Japanese sliding door pull
- Bottom rail (┴─┴ / ┻━┻) at ThresholdRow
- Threshold shadow (▔) below the door
- Compact fallback: existing fixed-layout rendering when height < 14

**Key mapping:** The existing Shoji mid-cross bar concept maps perfectly to DoorAnatomy's PanelDivider. The handle uses ○ (open circle) to suggest a recessed sliding door pull, matching Modern's minimalist handle choice.

**Testing:** Add height-aware unit tests and golden file tests (h16, h24, selected/unselected). Existing compact tests remain valid by passing height=0.

## Rejected Options

### Option A: Keep existing fixed layout, just scale vertically
Repeat empty pane rows to fill height, keeping the fixed position structure.
**Rejected because:** Doesn't use DoorAnatomy positions, so structural elements (panel divider, handle) won't be at consistent proportions across themes. Inconsistent with 35.2-35.4 approach.

### Option B: Multiple cross junctions for tall doors
Add additional ┼ junctions to fill vertical space and create a more grid-like appearance.
**Rejected because:** Course correction CC-theme-rendering-fixes already flagged "dense lattice creates visual noise" as a design issue. More cross junctions would worsen this. Keep it minimal with a single cross junction at PanelDivider.

### Option C: Use ● (filled circle) for handle
Consistent with Classic theme's doorknob.
**Rejected because:** ○ (open circle) better represents a Japanese sliding door's recessed pull handle. Each theme should have handle character appropriate to its aesthetic.
